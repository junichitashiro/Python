# 処理概要

クリックでクリップボードに情報をコピーするボタンを配置する

## 処理内容

* カレントディレクトリにあるExcelファイルを読み込む
* 読み込んだテキスト情報を持つボタンを作成する
* このボタンをクリックするとテキスト情報がクリップボードにコピーされる

### Excelファイル

* ファイル名：data.xlsx
* A,Bの2列×複数行で入力し、この行は可変とする

| A          | B                            |
| ---------- | ---------------------------- |
| Diggy-MO'  | メインMC・ボーカル           |
| Bro.Hi     | ヒューマンビートボックス・MC |
| Shinnosuke | キーボード・プログラミング   |
